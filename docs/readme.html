📘 Panduan Penggunaan — Simple Inventory Lite

📦 Apa Ini?

Simple Inventory Lite adalah aplikasi inventaris offline yang dibuat khusus untuk pelaku UMKM di Indonesia. Tidak perlu instalasi server, tinggal klik dan langsung bisa dipakai.

🚀 Cara Menjalankan

  1. Download paket dari link resmi kami (MEGA.nz atau halaman produk).
  2. Ekstrak file ZIP ke folder mana saja di komputer Anda.
  3. Klik dua kali simple-inv.vbs untuk langsung menjalankan aplikasi.
  4. Aplikasi akan membuka browser dan siap digunakan.

🖥️ Kebutuhan Sistem

  • Windows 7 / 10 / 11
  • Tidak perlu internet
  • Tidak perlu XAMPP / PHP / MySQL

🔐 Privasi & Data

Semua data disimpan secara lokal di komputer Anda menggunakan database SQLite. Tidak ada koneksi keluar. Privasi Anda 100% aman.
